About World Pulse

World Pulse is a daily observation layer for public anomaly and attention signals.

What Is World Pulse

World Pulse collects selected public datasets, normalizes them into observed events, and summarizes how unusual the selected data date appears relative to a recent baseline. It is designed for observation, not prediction.

Three Principles

  • No predictions.
  • No panic.
  • No financial, medical, political, or emergency-response advice.

World Pulse does not issue alerts, forecasts, warnings, or recommendations.

Data Window

World Pulse uses a data date rather than claiming to represent the exact current moment.

Some public datasets, especially daily aggregated sources such as Wikipedia Pageviews, may become available with a delay. For this reason, the displayed data date can lag behind the calendar date.

The interface uses "Signal Position" and "Data date" to avoid implying real-time coverage.

What "Signal Position" Is NOT

  • Not a risk level. World Pulse does not measure danger.
  • Not an alert. World Pulse does not issue warnings.
  • Not a forecast. World Pulse does not predict.
  • Not a recommendation. Do not use it for trading, medical, political, or personal safety decisions.
  • A high Signal Position does not mean something bad will happen.
  • A low Signal Position does not mean the world is safe.

Sources

  • USGS Earthquake Hazards Program
  • NOAA Space Weather Prediction Center
  • Wikipedia Pageviews
  • Open Notify
  • Cloudflare Radar internet outage/anomaly observations

Sources are stored as raw observations with source lineage where possible.

Methodology

World Pulse stores raw public observations, records source lineage, normalizes them into events, and compares selected signals against a recent baseline.

The current displayed Signal Position uses internal score version weirdness_v0_2.

weirdness_v0_2 computes a daily raw score from positive anomaly scores and converts the selected data date into a percentile rank within the recent observed window.

The current beta uses a 30-day baseline window where available. Some sources may have fewer observations due to API availability or collection start date.

World Pulse internally distinguishes reality, attention, and context layers for future research. Public displays remain a single Signal Position.

Top Signal and Quiet Signal

Top Signal highlights an individual observed event from the selected data date. Daily aggregation events, such as Wikipedia attention snapshots, may contribute to scoring context but are excluded from the displayed Top Signal.

Quiet Signal is an optional display-only field. It may show one individual observed signal above the recent baseline within the current data window. It is not a forecast, alert, warning, or recommendation, and it may be absent on many data dates.

License & Attribution
