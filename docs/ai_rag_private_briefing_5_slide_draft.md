## Slide 1: What World Pulse Reviews

### Main message

World Pulse compares selected public reality records with activity in structured reference systems.

### Slide text

- Source-recorded reality events
- Reviewed candidate reference pages
- Pageview windows around event dates
- Internal review of reference-data freshness windows

### Visual direction

Simple flow:

Reality event -> Candidate reference pages -> Pageview window

### Speaker note

World Pulse does not measure real-time attention. It reviews how selected source-recorded events relate to later activity in structured reference systems.

## Slide 2: Why This May Matter for AI/RAG

### Main message

AI/RAG systems depend on reference systems, and reference systems do not update evenly across domains or event types.

### Slide text

- AI/RAG workflows rely on reference systems.
- Reference systems change unevenly.
- Some event categories show clearer reference-system activity than others.
- World Pulse can help review candidate freshness windows.

### Visual direction

Small two-column layout:

| Reference systems | AI/RAG workflows |
| --- | --- |
| Reality events | Reference-system activity |

Small box:

Exploratory review, not model-quality diagnosis.

### Speaker note

This is not a claim about model quality. It is a method for reviewing selected reference-system activity around source-recorded events.
